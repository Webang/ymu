# TabbarItem


## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 | 版本说明 |
| - | - | - | - | - | - |
| title | 底部显示字符 | String | - | - | - |
| icon | 顶部显示的icon | String | - | - | - |
| selected | 是否选中 | Boolean | - | false | - |
| selectedIcon | 选中状态时显示的icon | - | - | - | - |
| badge | 违章 | Number, String | - | - | - |
| dot | 小圆点 | Boolean | - | - | - |
| color | 颜色 | String | - | true | - |
| selectedColor | 选中状态的颜色 | String | - | true | - |


## Slots

| 名字 | 说明 |
| - | - |
| badge | badge 插槽 |
| icon | icon 插槽 |
| title | title 插槽 |
| default | tabbarItem 插槽 |


## Events

| 名字 | 说明 |
| - | - |
| on-click-item | 点击事件 |
