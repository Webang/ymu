# Radio 单选框

```JS
import { Radio } from 'yum';
Vue.use(Radio);
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 |
| - | :-: | :- | :-: | :-: |
| label | value值 | String, Number, Boolean | - | - |
| value | 绑定 v-model | String |- | - |
| name | 右侧显示的文字 | String | - | - |
| disabled | 是否为禁用状态 | Boolean | - | - |


## Slots

| 名字 | 说明 |
| :-: | :-: |
| default | name 插槽 |
| icon | icon 插槽，方便自定义icon |


## Events

| 名称 | 描述 |
| :-: | :-: |
| input | - |
