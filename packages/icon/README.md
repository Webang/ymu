# 图标 Icon

使用默认 `ymucion` 字体图标

```JS
import Vue from 'vue'
import { Icon } from 'ymu'

Vue.use(Icon)
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 | 版本说明 |
| - | - | - | - | - | - |
| name | 图标名称 | String | - | - | - |

## Slots

无

## Events

| 名称 | 描述 |
| :-: | :-: |
| click | 点击图标 |

## Sass vars

无
