# CountdownNum 倒计时

```JS
import Vue from 'vue'
import CountdownNum from 'ymu'

Vue.use(CountdownNum)
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 | 版本说明 |
| - | - | - | - | - | - |
| count | 计数大小 | Number | - | - | - |
| delay | 计数延迟 | Number | - | - | - |
| play | 自动开始 | true | - | - | - |

## Slots

| 名字 | 说明 |
| :- | :- |
| - | - |

## Events

| 名称 | 描述 |
| :-: | :-: |
| on-change | 时间发生变化 |
| on-end | 时间已结束 |
