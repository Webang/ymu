# Backdrop 幕布

```JS
import Vue from 'vue';
import { Backdrop } from 'yus';

Vue.use(Backdrop);
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 |
| - | - | - | - | - |
| value | 显示幕布 | Boolean | - | - |

## Slots

| 名称 | 描述 |
| - | - |
| default | - |

## Events

| 名称 | 描述 | 回调参数 |
| - | - | - |
| click | 点击幕布时触发 | event |
