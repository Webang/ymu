# Clocker 时钟

```JS
import Vue from 'vue'
import { Clocker } from 'yum'

Vue.use(Clocker)
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 | 版本支持 |
| - | - | - | - | - | - |
| time | 当前时间 | Number(毫秒时间戳), String(标准日期格式) | - | - | - |
| startTime | 开始时间 | Number(秒时间戳) | - | - | - |
| endTime | 结束时间 | Number(秒时间戳) | - | - | - |

## Slots

| 名字 | 说明 |
| - | - |
| default | Checkbox 组件列表 |
