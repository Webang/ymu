# Spinner 加载动画

```JS
import Vue from 'vue';
import { Spinner } from 'f6-vue';

Vue.use(Spinner);
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 |
| - | - | - | - | - |
| type | 动画类型 | String | - | crescent |
| color | 颜色 | String | - | currentColor |
