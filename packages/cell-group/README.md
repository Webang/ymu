# CellGroup

```JS
import Vue from 'vue'
import { CellGroup } from 'yum'

Vue.use(CellGroup)
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 |
| - | :-: | :- | :-: | :-: |
| title | 顶部文字，复杂布局请使用slot | String | - | - |

## Slots

| 名字 | 说明 |
| :-: | :-: |
| title | title 插槽 |
| default | - |

## Events

无

$cell-group-herizontal-padding: $layout-herizontal-padding;
