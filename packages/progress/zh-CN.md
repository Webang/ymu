# Progress 进度条

```JS
import Vue from 'vue';
import { Progress } from 'f6-vue';

Vue.use(Progress);
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 | 版本说明 |
| - | - | - | - | - | - |
| percentage | 百分比 | Number | - | - | - |
| bar-height | 高度 | Number | - | - | - |

## Slots

| 名称 | 描述 |
| - | - |
| start | start 插槽 |
| end | end 插槽 |
