# Card 卡片

```JS
import Vue from 'vue';
import { Card } from 'f6-vue';

Vue.use(Card);
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 |
| - | - | - | - | - |
| title | 卡片标题 | String | - | - |
| footer | 卡片页脚 | String | - | - |

## Slots

| 名称 | 描述 |
| - | - |
| title | - |
| default | - |
| footer | - |

## Events

| 名称 | 描述 |
| - | - |
| click | 点击卡片时触发 |
