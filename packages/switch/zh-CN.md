# Switch 开关

```
import Vue from 'vue';
import { Switch } from 'ymu';

Vue.use(Switch);
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 | 版本说明 |
| - | - | - | - | - | - |
| v-model | 开关选中状态 | Boolean | - | - | - |
| disabled | 禁用状态 | Boolean | - | - | - |


## Slots

无

| - | - |
