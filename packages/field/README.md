# 输入框 Field

Confirm 通常和 Dialog 搭配使用，Dialog 通常他包含了3个部分：

- ym-dialog__hd
- ym-dialog__bd
- ym-dialog__ft

```JS
import Vue from 'vue'
import Confirm from 'ymu'

Vue.use(Confirm)
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 | 版本说明 |
| - | - | - | - | - | - |

| 名字 | 说明 |
| - | - |
