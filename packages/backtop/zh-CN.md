# Backtop 回到顶部

```JS
import Vue from 'vue'
import { Backtop } from 'yum'

Vue.use(Backtop)
```

## Props

| 属性 | 描述 | 类型 | 可选值 | 默认值 |
| - | :-: | :- | :-: | :-: |
| text | 显示文本 | String | - | 顶部 |

## Slots

| 名称 | 描述 |
| :-: | :-: |
| - | - |

## Events

| 名称 | 描述 |
| :-: | :-: |
| click | - |
